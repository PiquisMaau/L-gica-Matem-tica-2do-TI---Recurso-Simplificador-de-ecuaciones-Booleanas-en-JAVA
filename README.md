# Simplificador de Ecuaciones Booleanas en Java  
Proyecto desarrollado como recurso didáctico para la exposición del tema **Álgebra Booleana** en la asignatura de **Lógica Matemática**, 2do "A" – Tecnologías de la Información, Universidad Técnica de Ambato.

---

## Descripción del Proyecto
Este software es un **simplificador de ecuaciones booleanas**, implementado en Java, que permite procesar expresiones lógicas y aplicar reglas de simplificación para obtener una versión más reducida y eficiente de la expresión original.

El objetivo principal es servir como herramienta educativa para comprender mejor los principios del álgebra booleana y apoyar la exposición del Grupo 5.

---

## Funcionalidades Principales
- Lectura y procesamiento de expresiones booleanas.
- Aplicación de reglas de simplificación:
  - Identidad  
  - Nulidad  
  - Idempotencia  
  - Involución  
  - Complemento  
  - Absorción  
  - Distributiva  
  - De Morgan  
- Interfaz gráfica sencilla e intuitiva.
- Resultados claros y en tiempo real.
- Código modular y fácil de extender.

---

## Tecnologías Utilizadas
- **Java 8+**
- **Swing / AWT** (para la interfaz gráfica)
- **NetBeans** como entorno de desarrollo
- Programación orientada a objetos
